The main components of the project include:
- 3DR’s smart drone Solo.
- ADS-B Receiver.

The image shows how the components are organized. Specifically, the Solo smart-drone is equipped with four rotors, various sensors, and an on-board computer running a Linux installation. The drone is primarily controlled via 3DR’s DroneKit, an API that allows developers to create applications to monitor and control the drone.

Using the Solo’s accessory bay, we have attached an ADS-B receiver, which will be processed by the on-board computer using our software. The system will also communicate with Nevada Dynamic’s ground control to provide the user with useful feedback.
